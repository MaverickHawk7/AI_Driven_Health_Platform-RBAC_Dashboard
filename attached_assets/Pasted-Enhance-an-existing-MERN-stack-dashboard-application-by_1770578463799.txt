Enhance an existing MERN-stack dashboard application by adding basic authentication, data persistence, and a Field Worker workflow. The goal is to preserve existing frontend pages and user flows while implementing only the minimum backend logic required for a functional demo.

Do not redesign the UI. Do not implement real AI logic.

Tech Stack

Frontend: React (existing pages already present)

Backend: Node.js + Express

Database: MongoDB (Mongoose)

Auth: Email + password with JWT

No OTP, no refresh tokens, no real AI
