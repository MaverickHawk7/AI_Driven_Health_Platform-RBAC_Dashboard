Build and enhance this dashboard with basic authentication, role-based access, and a Field Worker workflow. This is an extension of an existing dashboard UI, not a rebuild.

Tech Stack (Mandatory)

Frontend: React

Backend: Node.js + Express

Database: MongoDB (Mongoose)

Authentication: JWT (email + password only)

No AI/ML implementation — simulate AI results
